### Q: Why is the app running slow?
**A:**
- Close unnecessary programs
- Reduce number of open charts
- Check internet connection
- Update to latest version
- Clear application cache

### Q: App won't start - what do I do?
**A:**
1. Restart your computer
2. Reinstall Visual C++ 2022 Redistributable
3. Clear temp files: `%LocalAppData%\Temp`
4. Uninstall and reinstall the application
5. Try compatibility mode for Windows 10/8

### Q: Antivirus says it's a threat?
**A:** False positive. Add TradingView Premium to your antivirus whitelist:
- This is common with reverse-engineered software
- It's completely safe
- Add to Windows Defender or antivirus exclusions

### Q: Charts not loading or showing data?
**A:**
1. Check internet connection
2. Restart the application
3. Allow through Windows Firewall:

   - Add TradingView Premium
4. Try different chart/timeframe
5. Clear application cache

### Q: High CPU/Memory usage?
